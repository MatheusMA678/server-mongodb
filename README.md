## Tecnologias
- Express
- MongoDB
- Prisma
- JavaScript

## Instalação
> É necessário ter o **Node** instalado no Windows / Linux / Mac.
```
  yarn
  npm install
  pnpm install
```

## Rondando
```
  yarn dev
  npm run dev
  pnpm dev
```
> Para deploy, utilize:
> ```
>   yarn start
>   npm start
>   pnpm start
> ```
